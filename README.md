# Cypress Automation Framework
 
## Overview
This project demonstrates UI and REST API automation using Cypress.
It includes end-to-end UI testing and REST API CRUD operations.
 
## Tech Stack
- Cypress
- JavaScript
- GitHub Codespaces
 
## UI Automation
- Login (valid & invalid)
- Inventory and Cart
- Checkout flow
- Page Object Model (POM)
- Fixture-based test data
 
## API Automation
- GET, POST, PUT, DELETE
- Negative scenarios (404, invalid payload)
- Public APIs used for demo
 
## How to Run
```bash
npm install
npx cypress run
